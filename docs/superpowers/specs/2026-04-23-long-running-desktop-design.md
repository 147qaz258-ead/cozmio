# 桌面端应用设计方案（修订版）

> **目标**：将 cozmio 打造成长期驻留运行的桌面智能体

---

## 核心设计原则

**三者解耦**：

| 概念 | 含义 | 独立于 |
|------|------|--------|
| **应用驻留** | 进程常驻后台，终端关闭不退出 | 不依赖窗口状态 |
| **运行状态** | 监控循环是否工作（Running / Stopped） | 不依赖窗口是否显示 |
| **窗口显示** | 主界面可见性 | 不依赖运行状态 |

**主界面职责**：查看状态 + 查看内容（最近记录）+ 启动/停止控制
**托盘职责**：常驻入口 + 快速控制

---

## 状态定义

```rust
enum AppRunningState {
    Running,  // 监控循环工作
    Stopped,   // 监控循环停止
}
```

注意：原有 `TrayState` 包含 Processing 等，用于内部状态；对外暴露用 `AppRunningState`。

---

## 主窗口

**显示逻辑**：
- 应用启动时默认 `visible: false`（隐藏）
- 可通过托盘菜单「打开主界面」打开
- 点 X 按钮 → 隐藏窗口，不退出应用
- 窗口关闭 ≠ 应用退出

**窗口内容**：

```
┌────────────────────────────────────────┐
│  Cozmio - 主动智能体                    │
├────────────────────────────────────────┤
│  [状态指示] ● Running / ○ Stopped      │
│                                        │
│  ── 最近记录 ──────────────────────    │
│  │ 10:32:15 │ CONTINUE │ Chrome      │  │
│  │ 10:32:08 │ ABSTAIN  │ VSCode     │  │
│  │ 10:31:55 │ CONTINUE │ Chrome     │  │
│  └────────────────────────────────────│  │
│                                        │
│  [ 启动 ]  [ 停止 ]                    │
└────────────────────────────────────────┘
```

- 顶部：状态指示（Running/Stopped）
- 中部：最近行为记录列表（只读，滚动查看）
- 底部：两个操作按钮

**按钮语义**：
- **启动**：设置状态为 Running，监控循环开始工作
- **停止**：设置状态为 Stopped，监控循环停止（但应用继续驻留）

---

## 托盘菜单

| 菜单项 | 功能 |
|--------|------|
| 打开主界面 | 显示主窗口 |
| 状态：Running / Stopped | 显示当前状态（不可点击） |
| 启动 | 设置 Running 状态 |
| 停止 | 设置 Stopped 状态 |
| 打开日志目录 | 用 explorer 打开 `%LOCALAPPDATA%\cozmio\` |
| 退出 | 正常退出应用 |

---

## 应用行为矩阵

| 事件 | 应用驻留 | 运行状态 | 窗口显示 |
|------|----------|----------|----------|
| 系统启动 | ✅ 常驻 | Stopped | ❌ 隐藏 |
| 点击"启动"按钮 | ✅ 常驻 | Running | 不变 |
| 点击"停止"按钮 | ✅ 常驻 | Stopped | 不变 |
| 点击"打开主界面" | ✅ 常驻 | 不变 | ✅ 显示 |
| 点击窗口 X 按钮 | ✅ 常驻 | 不变 | ❌ 隐藏 |
| 点击托盘"退出" | **退出** | - | - |

## 单实例保证

使用 Tauri single-instance plugin，应用启动时检测是否已有实例：
- 若已有实例 → 聚焦已有窗口，当前实例退出
- 若没有实例 → 正常启动

---

## 开机自启（可选，可配置）

使用 Tauri autostart plugin：
- 默认关闭
- 可在设置中开启
- 开启后写入 Windows Registry Run key

## 文件结构

```
cozmio/
├── src-tauri/
│   ├── src/
│   │   ├── main.rs           # 应用入口，单实例配置
│   │   ├── main_loop.rs      # 监控循环（依赖运行状态）
│   │   ├── tray.rs           # 托盘菜单（启动/停止/退出）
│   │   ├── commands.rs       # Tauri IPC 命令
│   │   ├── window.rs         # 主窗口控制（显示/隐藏）
│   │   └── ...
│   └── tauri.conf.json       # visible: false, singleInstance
```

---

## 实现要点

1. **主窗口 visible: false** - 默认隐藏
2. **X 按钮 → hide()** - 不调用 app.exit()
3. **运行状态通过 TrayState/标志位控制** - main_loop 读取状态决定是否工作
4. **单实例** - tauri-plugin-single-instance
5. **退出走正常收尾** - main_loop 收到退出信号后优雅终止
