## 一、项目概述

### 1.1 项目名称

**Cozmio**（酷泽缪）

### 1.2 一句话简介

Cozmio 是运行在用户桌面端的 AI Agent 伴侣——静默监控工作场景，智能判断介入时机，通过 Relay 网络调度合适 Agent 完成任务，让人机协作无缝闭环。

### 1.3 详细描述

当前 AI Agent 市场以云端服务为主，用户需要主动打开对话界面、粘贴上下文、等待结果返回。人机协作体验割裂，AI 能力与桌面工作流分离。

**Cozmio 的核心创新**：把 AI 能力植入操作系统层。

它像一位安静的桌面伴侣，运行在用户 Windows 电脑的系统托盘里：
- **静默监控**：每秒检测前台窗口变化，不打扰用户
- **智能判断**：调用本地 Ollama 视觉模型，理解当前工作场景
- **按需介入**：仅当高价值任务（如代码编写、文档编辑）需要帮助时，通过 Windows Toast 通知询问用户
- **Relay 调度**：用户确认后，Relay 网络匹配并调度最合适的 Agent 执行任务
- **结果闭环**：任务完成后推送结构化结果到桌面，完整的人机协作闭环

**技术架构亮点**：
- 本地优先，数据不离开用户电脑
- 支持 Box H1 硬件（树莓派 + Qwen2-VL-4B 模型）作为边缘推理节点
- Practice Loop 持续学习用户偏好，优化判断质量

### 1.4 核心功能列表

| 功能模块 | 说明 | 状态 |
|---------|------|------|
| Window Monitor | 前台窗口实时监控与截图 | ✅ 已上线 |
| Vision Model Judgment | 本地 Ollama 视觉模型判断是否介入 | ✅ 已上线 |
| Notification Chain | Windows Toast 通知 + cozmio:// 协议回调 | ✅ 已上线 |
| Relay Network Dispatch | 去中心化 Agent 调度网络 | ✅ 已上线 |
| Practice Loop | 用户反馈驱动的持续学习机制 | ✅ 已上线 |
| Memory Competition | 多记忆源竞争，生成最佳上下文 | ✅ 已上线 |
| Desktop Node (Box H1) | 边缘硬件推理节点，支持视觉多模态 | ✅ 已上线 |

## 二、市场分析

### 2.1 行业背景

**全球 AI Agent 市场正在爆发**：
- 2024 年全球 AI Agent 市场规模约 55 亿美元，预计到 2030 年将超过 500 亿美元（CAGR > 45%）
- 主要玩家包括 Anthropic Claude Agent、OpenAI GPT Operator、Microsoft Copilot Agent
- 但这些产品均为云端 SaaS 形态，需用户主动使用，与桌面工作流割裂

**桌面端 AI 介入的空白**：
- 现有 AI 工具要求用户"打开 AI 界面 → 粘贴上下文 → 等待结果 → 复制回桌面"
- 没有一款产品真正"看"到用户的桌面工作，感知当前任务场景
- 操作系统层面的 AI 介入是一片蓝海

### 2.2 目标用户

**核心用户群体**：

| 用户类型 | 特征 | 需求 |
|---------|------|------|
| 开发者 | 在 IDE 中编写代码 | 代码补全、调试辅助、重构建议 |
| 内容创作者 | 编辑文档/写报告 | 写作建议、内容润色、资料整理 |
| 数据分析师 | 处理表格/报告 | 数据清洗、可视化建议、分析解读 |
| 研究人员 | 查阅文献/写论文 | 文献检索、摘要生成、引用检查 |

**用户场景举例**：
- "我在 VS Code 写代码时，Cozmio 自动检测到我在实现一个复杂算法，弹出通知问我是否需要 Agent 辅助——我确认后，Relay 网络调度了一个擅长代码重构的 Agent，5分钟后结果返回到我的桌面"
- "我在 Excel 处理销售数据，Cozmio 发现数据模式异常，通知我是否需要数据分析 Agent——我确认后，获得了完整的分析报告和数据可视化建议"

### 2.3 竞争分析

| 竞品 | 类型 | 优势 | 劣势 | Cozmio 差异点 |
|------|------|------|------|---------------|
| Claude Agent | 云端 AI 助手 | 模型能力强，品牌知名 | 需主动使用，数据在云端 | 本地运行，感知桌面场景 |
| GitHub Copilot | IDE 插件 | 与开发工具深度集成 | 仅限开发场景 | 全场景桌面感知 |
| Microsoft Copilot | 操作系统集成 | Windows 原生支持 | 功能受限，企业导向 | 更灵活的个人助理 |
| OpenAI Operator | 浏览器自动化 | 自动化能力强 | 仅限网页场景 | 桌面全场景覆盖 |

**Cozmio 竞争优势**：
1. **本地优先**：数据不离开用户电脑，隐私安全
2. **场景感知**：通过窗口监控真正"看到"用户在工作什么
3. **Relay 网络**：去中心化调度，不被单一模型提供商绑定
4. **边缘计算**：支持 Box H1 硬件，隐私敏感场景可在本地推理
